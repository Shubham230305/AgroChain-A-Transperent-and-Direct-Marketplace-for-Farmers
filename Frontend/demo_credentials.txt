### Test Instructions

1. **Login Testing:**
   - Use any valid email/phone with matching password
   - Try invalid combinations to see error messages

2. **Signup Testing:**
   - Fill all fields with valid data
   - Test validation with invalid inputs
   - Password must be at least 6 characters
   - Aadhaar must be exactly 12 digits
   - Phone must be exactly 10 digits starting with 6-9

3. **Language Testing:**
   - Switch between English, Marathi, and Hindi
   - All UI text will update automatically

4. **Forgot Password:**
   - Enter any email or phone number
   - System will simulate sending reset link

### Notes:
- This is a demo application - no real authentication is performed
- All login/signup attempts will succeed with valid format
- The system validates format but doesn't check against real databases
- After successful login, you'll be redirected to the dashboard
